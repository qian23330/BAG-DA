# BAG-DA
 A tool of brain age gap(BAG)-disease risk associations calculation
## Python packages required
 Python version: 3.11.5 \
 Ensure you have installed all python packages of version below:
 - pandas 2.1.2
 - numpy 1.24.3
 - scikit-learn 1.3.0
 - scipy 1.11.3
 - statsmodels 0.14.0
 - lifelines 0.28.0
 - pydeseq2 0.5.0
